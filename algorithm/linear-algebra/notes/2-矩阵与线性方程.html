      # 矩阵与线性方程


      作者：Seiya

      时间：2019年07月25日


      # 引言


      矩阵是高等代数学中的常见工具，也常见于统计分析等应用数学学科中。矩阵的运算是数值分析领域的重要问题。将矩阵分解为简单矩阵的组合可以在理论和实际应用上简化矩阵的运算。




      # 矩阵定义


      将一些元素排列成若干行，每行放上相同数量的元素，就是一个矩阵。如以下的矩阵排列成的形状是矩形，所以称为矩阵。

      # 矩阵的标记


      一个矩阵 A\mathbf AA 从左上角数起的第 iii 行第 jjj 列上的元素称为第 i,ji,ji,j 项，通常记为 Ai,j\mathbf {A} _{i,j}A​i,j​​、Aij\mathbf {A} _{ij}A​ij​​ 或 A[i,j]\mathbf {A} _{[i,j]}A​[i,j]​​。如果不知道矩阵 A\mathbf AA 的具体元素，通常也会将它记成 A[aij]m×n\mathbf A [\mathbf a _{ij}] _{m \times n}A[a​ij​​]​m×n​​。反之，如果 A\mathbf AA 的元素可以写成只与其行数 iii 和列数 jjj 有关的统一函数 fff，那么也可以用 A=[f(i,j)]m×n\mathbf A = \left[ f(i,j) \right] _{m \times n}A=[f(i,j)]​m×n​​ 作为 A\mathbf AA。

      # 可逆矩阵


      逆矩阵（ inversematrixinverse matrixinversematrix ），又称反矩阵。在线性代数中，给定一个 n 阶方阵 A\mathbf {A}A，若存在一 n 阶方阵 B\mathbf {B}B，使得 AB=BA=In\mathbf {AB} = \mathbf {BA} = \mathbf {I} _{n}AB=BA=I​n​​，其中 In\mathbf {I} _{n}I​n​​ 为 n 阶单位矩阵，则称 A\mathbf {A}A 是可逆的，且 B\mathbf {B}B 是 A\mathbf {A}A 的逆矩阵，记作 A−1\mathbf {A} ^{-1}A​−1​​。

      注意：

      只有方阵（n × n 的矩阵）才可能有逆矩阵。若方阵 A\mathbf {A}A 的逆矩阵存在，则称 A\mathbf {A}A 为非奇异方阵或可逆方阵。



      # 线性方程组与逆矩阵


      若线性方程组 Ax=b\mathbf {A} x = bAx=b，对任意向量 b 有唯一解，则 A\mathbf {A}A 是可逆的( invertibleinvertibleinvertible )。

      对线性方程组 Ax=b\mathbf {A} x = bAx=b，若 A\mathbf {A}A 可逆，则可由常数项 b 求得 x。
