      # 引言


      线性代数的中心问题是求解线性方程组。

      线性代数是建立在向量的加法和数乘这两种所谓“线性运算”上的。



      # n 维空间中的点


      在直线上取定坐标系后，每个实数一一对应表示直线上的点。一个二元有序数组（x, y）一一对应地表示平面上的点。一个三元有序数组（x, y, z）一一对应地表示三维空间中的点。同理，定义 n 维空间的点为一个 n 元有序数组 x=(x1,...,xn)x = (x_1, ... , x_n)x=(x​1​​,...,x​n​​)，我们称 xix_ix​i​​ 为点 x 的第 iii 个坐标。


      # 加法与数乘运算

      在 n 维空间中，我们说两个点 x=(x1,...,xn)x = (x_1, ... , x_n)x=(x​1​​,...,x​n​​) 与 y=(y1,...,yn)y = (y_1, ... , y_n)y=(y​1​​,...,y​n​​) 相等，是指它们的每一个坐标相等，即：xi=yix_i = y_ix​i​​=y​i​​，∀1≤i≤n\forall 1 \leq i \leq n∀1≤i≤n。这是一个自然的定义（逻辑自洽的定义）。


      • 和运算

        可以定义两个点 x=(x1,...,xn)x = (x_1, ... , x_n)x=(x​1​​,...,x​n​​) 与 y=(y1,...,yn)y = (y_1, ... , y_n)y=(y​1​​,...,y​n​​) 的和为：x+y=(x1+y1,...,xn+yn)x + y = (x_1 + y_1, ... , x_n + y_n)x+y=(x​1​​+y​1​​,...,x​n​​+y​n​​)。


      • 数乘运算

        若 ccc 是一个数，可以定义数乘运算：cx=(cx1,cx2,...,cxn)cx = (cx_1, cx_2, ... , cx_n)cx=(cx​1​​,cx​2​​,...,cx​n​​)。


      # 点的运算性质

      容易看到这样定义的加法和数乘运算对任意 nnn 维空间中的点 (x, y, z) 满足一下 8 条运算性质：

      • 加法结合律

        比如：(x+y)+z=x+(y+z)(x + y) + z = x + (y + z)(x+y)+z=x+(y+z)

      • 加法交换律

        比如：x+y=y+xx + y= y + xx+y=y+x

      • 存在加法零元

        比如：若令 o=(0,...,0)o = (0,...,0)o=(0,...,0) 为一点，则：o+x=x+o=xo + x = x + o = xo+x=x+o=x

      • 对任何一个点存在负元素

        比如：若以 −x-x−x 表示 (−1)x(-1)x(−1)x 则：x+(−x)=0x + (-x) = 0x+(−x)=0

      • 1∗x=x1 * x = x1∗x=x

      • 乘法结合律

        比如：(c1∗c2)x=c1(c2∗x)(c_1 * c_2)x = c_1(c_2 * x)(c​1​​∗c​2​​)x=c​1​​(c​2​​∗x)

      • 加法分配律

        比如：(c1+c2)∗x=c1x+c2x(c_1 + c_2) * x = c_1x + c_2x(c​1​​+c​2​​)∗x=c​1​​x+c​2​​x

      • 数乘分配律

        比如：c∗(x+y)=cx+cyc * (x + y) = cx + cyc∗(x+y)=cx+cy



      # 向量


      # 定义

      向量指空间中具有一定长度及方向的直线段。

      从计算机专业的视角来看：向量指 n 个数 a1,...,ana_1, ... , a_na​1​​,...,a​n​​ 组成的有序数组 (a1,a2,...,an)(a_1, a_2, ..., a_n)(a​1​​,a​2​​,...,a​n​​)

      tips:

      注意：若两个向量相等 ⇔\Leftrightarrow⇔ 二者长度相等，方向相同。

      # 向量空间


      # 公理化定义

      给定数域 FFF，FFF 上的向量空间 VVV 是一个集合，其上定义了两种二元运算：

      • 向量加法: V × V → V，把 VVV 中的两个元素 u 和 v 映射到 VVV 中另一个元素，记作 u + v；

      • 标量乘法: F × V → V，把 FFF 中的一个元素 aaa 和 VVV 中的一个元素 u 变为 VVV 中的另一个元素，记作 aaa · u。

      VVV 中的元素称为向量，相对地，FFF 中的元素称为标量。


      而集合 VVV 公理才构成一个向量空间（也就是对 FFF 中的任意元素 a、b 以及 VVV 中的任意元素 u、v、w 都成立）：

      公理 说明
      向量加法的结合律 u+(v+w)=(u+v)+wu + (v + w) = (u + v) + wu+(v+w)=(u+v)+w
      向量加法的交换律 u+v=v+uu + v = v + uu+v=v+u
      向量加法的单位元 存在一个叫做零向量的元素 0 ∈ VVV，使得对任意u ∈ V都满足 u + 0 = u
      向量加法的逆元素 对任意v ∈ VVV 都存在其逆元素 −v ∈ VVV 使得 v + (−v) = 0
      标量乘法与标量的域乘法相容 a(bv)=(ab)va(bv) = (ab)va(bv)=(ab)v
      标量乘法的单位元 域 FFF 存在乘法单位元 1 满足 1v = v
      标量乘法对向量加法的分配律 a(u+v)=au+ava(u + v) = au + ava(u+v)=au+av
      标量乘法对域加法的分配律 (a+b)v=av+bv(a + b)v = av + bv(a+b)v=av+bv

      所有满足这些条件的集合 VVV 都可以被称为定义在数域 FFF 上的向量空间。


      tips:

      所谓向量空间就是一个满足一些特殊条件的集合，集合元素可以是任意的对象。


      # 基本性质

      以下是一些可以从向量空间的公理直接推出的性质：

      • 零向量 0 是唯一的；

      • 对任意 a ∈ FFF，a · 0 = 0；

      • 对任意 uuu ∈ VVV，0 · uuu = 0（0 是 FFF 的加法单位元）。

      • 如果 a · uuu = 0，则要么a = 0，要么 uuu = 0。

      • 向量加法的逆向量 vvv 是唯一的，记作 −v。u + (− v) 也可以写成 u − v，两者都是标准的。

      • 对任意 uuu ∈ VVV，−1 · uuu = −u.

      • 对任意 a ∈ FFF 以及 uuu ∈ VVV， (−a) · uuu = −(a · uuu) = a · (−u).



      # 向量间的线性关系


      # 线性组合

      给定 b,a1,a2,...,anb, a_1, a_2, ... , a_nb,a​1​​,a​2​​,...,a​n​​ 是 m 维向量（维度一致），若存在 k1,k2,...,knk_1, k_2, ... , k_nk​1​​,k​2​​,...,k​n​​，使：b=k1a1+k2a2+…+knanb = k_1a_1 + k_2a_2 + \ldots + k_na_nb=k​1​​a​1​​+k​2​​a​2​​+…+k​n​​a​n​​ 成立，就称 bbb 是 aaa 线性组的线性组合，也可以叫线性表示。相对的，k1,k2,...,knk_1, k_2, ... , k_nk​1​​,k​2​​,...,k​n​​ 被称为系数。


      • 基本性质：

        • 零向量可由任意向量组表示；

        • 向量组中任一向量可由向量组表示；

        • 任意向量可由 ϵ1=(1,0,0)\epsilon_1 = (1, 0, 0)ϵ​1​​=(1,0,0)，ϵ2=(0,1,0)\epsilon_2 = (0, 1, 0)ϵ​2​​=(0,1,0)，ϵ3=(0,0,1)\epsilon_3 = (0, 0, 1)ϵ​3​​=(0,0,1) 表示（这三个向量也被称为单位向量）；


      在三维空间中，一般而言，向量 u⃗\vec u​u​⃗​​，u⃗\vec u​u​⃗​​ 和 v⃗\vec v​v​⃗​​，或 u⃗\vec u​u​⃗​​ 、v⃗\vec v​v​⃗​​ 和 w⃗\vec w​w​⃗​​ 的所有线性组合分别是一条直线，一张平面或整个三维空间。


      # 向量组的等价

      给定两个向量组 α1,α2,…,αm\alpha_1, \alpha_2, \ldots , \alpha_mα​1​​,α​2​​,…,α​m​​ 和 β1,β2,…,βn\beta_1, \beta_2, \ldots , \beta_nβ​1​​,β​2​​,…,β​n​​，若这两个向量组同维（或两个向量组相互表示），就称作向量组的等价，即：{α1,α2,…,αm\alpha_1, \alpha_2, \ldots , \alpha_mα​1​​,α​2​​,…,α​m​​} ≅\cong≅ {β1,β2,…,βn\beta_1, \beta_2, \ldots , \beta_nβ​1​​,β​2​​,…,β​n​​}


      • 基本性质：

        • 反身性：一个向量组和自己本身等价{α1,α2,…,αm\alpha_1, \alpha_2, \ldots , \alpha_mα​1​​,α​2​​,…,α​m​​} ≅\cong≅ {α1,α2,…,αm\alpha_1, \alpha_2, \ldots , \alpha_mα​1​​,α​2​​,…,α​m​​}

        • 对称性：若 {α1,α2,…,αm\alpha_1, \alpha_2, \ldots , \alpha_mα​1​​,α​2​​,…,α​m​​} ≅\cong≅ {β1,β2,…,βn\beta_1, \beta_2, \ldots , \beta_nβ​1​​,β​2​​,…,β​n​​}，那么 {β1,β2,…,βn\beta_1, \beta_2, \ldots , \beta_nβ​1​​,β​2​​,…,β​n​​} ≅\cong≅ {α1,α2,…,αm\alpha_1, \alpha_2, \ldots , \alpha_mα​1​​,α​2​​,…,α​m​​}

        • 传递性：若 {α1,α2,…,αm\alpha_1, \alpha_2, \ldots , \alpha_mα​1​​,α​2​​,…,α​m​​} ≅\cong≅ {β1,β2,…,βn\beta_1, \beta_2, \ldots , \beta_nβ​1​​,β​2​​,…,β​n​​}，{β1,β2,…,βn\beta_1, \beta_2, \ldots , \beta_nβ​1​​,β​2​​,…,β​n​​} ≅\cong≅ {γ1,γ2,…,γs\gamma_1, \gamma_2, \ldots , \gamma_sγ​1​​,γ​2​​,…,γ​s​​}，那么 {α1,α2,…,αm\alpha_1, \alpha_2, \ldots , \alpha_mα​1​​,α​2​​,…,α​m​​} ≅\cong≅ {γ1,γ2,…,γs\gamma_1, \gamma_2, \ldots , \gamma_sγ​1​​,γ​2​​,…,γ​s​​}


      # 线性相关与线性无关重点

      • 线性相关

        对于 n 个 m 维向量 α1⃗,α2⃗,…,αn⃗\vec{\alpha_1}, \vec{\alpha_2}, \ldots , \vec{\alpha_n}​α​1​​​⃗​​,​α​2​​​⃗​​,…,​α​n​​​⃗​​，如果存在一组不全为零的 n 个数 k1,k2,…,knk_1, k_2, \ldots , k_nk​1​​,k​2​​,…,k​n​​，使得 ∑i=1nkiα⃗i=0⃗\sum_{i=1}^n {k_i \vec{\alpha}_i}=\vec{0}∑​i=1​n​​k​i​​​α​⃗​​​i​​=​0​⃗​​，称 n 个向量 α1⃗,α2⃗,…,αn⃗\vec{\alpha_1}, \vec{\alpha_2}, \ldots , \vec{\alpha_n}​α​1​​​⃗​​,​α​2​​​⃗​​,…,​α​n​​​⃗​​ 线性相关或线性相依。


      • 线性无关

        如果这样不全为零的 n 个数不存在，即上述向量等式仅当 k1=k2=…=kn=0k_1 = k_2 = \ldots = k_n = 0k​1​​=k​2​​=…=k​n​​=0 时才能成立，就称向量 α⃗1,α⃗2,…,α⃗n\vec{\alpha}_1, \vec{\alpha}_2, \ldots , \vec{\alpha}_n​α​⃗​​​1​​,​α​⃗​​​2​​,…,​α​⃗​​​n​​ 线性无关或线性独立。


      • 基本结论

        • 向量组中任意两向量成比例，必线性相关；

        • 含零向量的任意向量组必线性相关；

        • 单个零向量必线性相关；

        • 单个非零向量必线性无关（证明：α⃗≠0\vec\alpha \neq 0​α​⃗​​≠0， kα⃗=0⇒k=0k\vec\alpha = 0 \Rightarrow k = 0k​α​⃗​​=0⇒k=0）；

        • 单个向量 α⃗\vec\alpha​α​⃗​​ 线性相关 ⇔α⃗=0\Leftrightarrow \vec\alpha = 0⇔​α​⃗​​=0 ；

        • 若 α⃗1,…,α⃗n\vec\alpha_1, \ldots , \vec\alpha_n​α​⃗​​​1​​,…,​α​⃗​​​n​​ 线性相关，则 α⃗1,…,α⃗n,α⃗n+1,…,α⃗s\vec\alpha_1, \ldots , \vec\alpha_n, \vec\alpha_{n+1}, \ldots , \vec\alpha_s​α​⃗​​​1​​,…,​α​⃗​​​n​​,​α​⃗​​​n+1​​,…,​α​⃗​​​s​​ 线性相关；

          • 部分向量组线性相关 ⇒\Rightarrow⇒ 整体向量组线性相关；

          • 部分向量组线性无关 ⇒\Rightarrow⇒ 整体向量组线性无关；

        • n 个 n 维向量构成的行列式 D≠0⇔D \neq 0 \LeftrightarrowD≠0⇔ 线性无关，若 D=0⇔D = 0 \LeftrightarrowD=0⇔ 线性相关；


      • 定理

        • α⃗1,…,α⃗n\vec\alpha_1, \ldots , \vec\alpha_n​α​⃗​​​1​​,…,​α​⃗​​​n​​ 线性相关 ⇔\Leftrightarrow⇔ 至少一个向量可由其余向量线性表示；

        • α⃗1,…,α⃗n\vec\alpha_1, \ldots , \vec\alpha_n​α​⃗​​​1​​,…,​α​⃗​​​n​​ 线性无关，α⃗1,…,α⃗n,β\vec\alpha_1, \ldots , \vec\alpha_n, \beta​α​⃗​​​1​​,…,​α​⃗​​​n​​,β 线性相关，β\betaβ 可由 α⃗1,…,α⃗n\vec\alpha_1, \ldots , \vec\alpha_n​α​⃗​​​1​​,…,​α​⃗​​​n​​ 唯一表示；

        • 替换定理：α⃗1,…,α⃗s\vec\alpha_1, \ldots , \vec\alpha_s​α​⃗​​​1​​,…,​α​⃗​​​s​​ 线性无关，可由 β⃗1,…,β⃗t\vec\beta_1, \ldots , \vec\beta_t​β​⃗​​​1​​,…,​β​⃗​​​t​​ 表示，则 s<ts < ts<t；

          • 推论一：α⃗1,…,α⃗s\vec\alpha_1, \ldots , \vec\alpha_s​α​⃗​​​1​​,…,​α​⃗​​​s​​ 可由 β⃗1,…,β⃗t\vec\beta_1, \ldots , \vec\beta_t​β​⃗​​​1​​,…,​β​⃗​​​t​​ 表示，s>ts > ts>t，则 α⃗1,…,α⃗s\vec\alpha_1, \ldots , \vec\alpha_s​α​⃗​​​1​​,…,​α​⃗​​​s​​ 线性相关；

          • 推论二：m 个 n 为向量，且 m > n ，则必定线性相关；

          • 推论三：n + 1 个 n 维向量必定相关；

          • 推论四：等价的线性无关向量组，向量个数一定相同；


      # 线性相关与线性组合的区别重点

      线性组合 ↔\leftrightarrow↔ 方程有解，不是组合 ↔\leftrightarrow↔ 方程无解；

      线性相关 ⇔\Leftrightarrow⇔ 方程有非零解，线性无关 ⇔\Leftrightarrow⇔ 方程只有零解；



      # 向量的点积、长度


      # 向量的点积

      设 v⃗=(v1,…,vn)\vec v = (v_1, \ldots, v_n)​v​⃗​​=(v​1​​,…,v​n​​)，w⃗=(w1,…,wn)\vec w = (w_1, \ldots, w_n)​w​⃗​​=(w​1​​,…,w​n​​) 是两个 nnn 维向量，定义点积 v⃗⋅w⃗\vec v \cdot \vec w​v​⃗​​⋅​w​⃗​​ 为：v⃗⋅w⃗=v1w1+…+vnwn\vec v \cdot \vec w = v_1w_1 + \ldots + v_nw_n​v​⃗​​⋅​w​⃗​​=v​1​​w​1​​+…+v​n​​w​n​​。


      tips

      点积又被称为内积或数量积。两个向量的点积是一个数。


      # 向量的长度

      向量 v⃗\vec v​v​⃗​​ 的长度或模定义为：若 v⃗=(v1,…,vn)\vec v = (v_1, \ldots, v_n)​v​⃗​​=(v​1​​,…,v​n​​)，则 ∥v∥=v12+…+vn2\parallel v \parallel = \sqrt{v_1^2 + \ldots + v_n^2}∥v∥=√​v​1​2​​+…+v​n​2​​​​​。

      # 单位向量

      若 ∥v∥=1\parallel v \parallel = 1∥v∥=1，则 vvv 称为单位向量。
