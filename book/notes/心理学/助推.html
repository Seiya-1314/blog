    # 《助推 - 如何做出有关健康、财富与幸福的最佳决策》笔记


    作者：Seiya

    时间：2018年03月17日


    # 人类如何思考？


    # 1. 两种思维系统

    • 直觉思维系统：来自于人们的情感反应

      速度快、无意识、熟练

      不受控、不费力、联想


    • 理性思维系统：来自于人们有意识的思维

      速度慢、有意识，依靠规则

      受控、费力、演绎


    【举例】：

1. 青少年驾车比成年人驾车危险的原因之一便是青少年的直觉思维系统没有接受过太多的练习，而他们的理性思维
   系统又运行得太慢。

2. 《星际迷航》（Star Trek）中的斯波克先生（理性思维系统）和霍默·辛普森（直觉思维系统）。


    # 2. 三项经验法则

    • 锚定法则

    锚定一个已知数字的基础上沿着你认为正确的方向进行调整，然而我们做出的调整往往是不够的。


    • 可得性法则

    根据相关例子获得的难易程度来对风险的可能性做出判断。

    一个普遍存在的问题在于，越容易回忆起来的事件其实越容易夸大人们对可能性的判断；相反，对于很少在记忆中保留下来的事情，人们便会认为它们发生的可能性很小。

    【案例】：

1、哮喘病比龙卷风更可怕，新闻里面经常会出现龙卷风的灾害有多严重，但很少出现哮喘病的危害；

2、如果最近一段时间没有发生过水灾，那么生活在洪涝区域的人们便会很少去购买有关的保险。


    • 代表性法则

    一种相似性启发

    【举例】：

1、对于一名身高近2米的非洲裔美国人和一名身高1.7米的犹太人，我们更倾向于将前者看作是一名职业篮球运动
   员，因为篮球运动员大多身材高大，很少有矮个子参与（特别是在近年）

2、篮球“手热效应”，即连续命中效应；

3、癌症扎堆现象，所谓癌症扎堆现象可能只是一种随机的波动而已，多数情况下，这种担心没有必要；

4、人们发现异常数据时，总是倾向于是有某种原因造成的，实际上，也许只是正常波动而已；


    # 3. 盲目乐观

    人类所普遍存在的一种心理状态。由于他们过高估计自己对危害性的免疫能力，所以无法采取任何理性的预防措施。

    不切实际的乐观可以帮助我们解释人们的许多冒险行为，特别是在危及生命和健康的领域。

    【举例】：

1、当学生对将来进行展望时，他们通常都会认为自己较之其他学生更不容易在将来的工作中被炒鱿鱼，更不容易
   得心脏病或癌症，更不容易离婚，也更不容易出现酗酒问题；

2、男性同性恋者会普遍低估自己染上艾滋病的概率，尽管他们知道艾滋病的存在；

3、年纪较大的人会低估自己遭遇车祸或者罹患重大疾病的概率；

4、烟民都知道吸烟有害健康，他们甚至会过分夸大吸烟的危害，但他们中绝大部分人却不太相信自己比大部分不
   吸烟的人更容易患肺癌和心脏病。

5、彩票之所以热销，利用的也是人们不切实际的乐观心理。


    # 4. 损失厌恶

    大体上来讲，失去某件东西使你难过的程度比你得到这东西使你快乐的程度大两倍。

    损失厌恶容易使人产生惰性，即一种强烈的保持现状的欲望。因为不想招致损失而不愿意放弃某些事情，拒绝因此发生的交易。

    【举例】：

1、我问你是否愿意打赌掷硬币，正面朝上你会赢一些钱，反面朝上你会输100元。那么“这一些钱”为多少时，你才
   会心动，对大多数人来说，答案大约是200元左右。这意味着赢200元的幸福刚好能够抵消失去100元的懊恼。

2、在一个实验中，一半学生得到咖啡杯，一半学生得到一大块巧克力，咖啡杯的价值和巧克力的价值等同。在得到
   两样东西之前，学生们表现出来对二者基本相等的占有欲望。然而一旦他们拿到了各自的东西，当再次要求他们
   将手中的东西换成另外一种时，仅有10%的人愿意这样做。


    # 5. 现状偏见

    “现状偏见”的原因之一是用心程度不够。

    许多人都会采取一种“既来之，则安之”的态度。

    【举例】：

1、出版社免费赠送三个月杂志，三个月后照价付款杂志，直到最终设法取消预订;

2、看电视时的延滞效应，网络公司会花费大量时间考虑节目安排计划，因为他们知道，晚上打开NBC节目的观众倾
   向于不换台而停在那里。当一个节目播放结束，下一个节目即将播出的时候，多数观众会想“既来之，则安之”;


    # 6. 措辞优化

    公共节能领域的两条宣传语：

    第一，如果你采取节能措施，你将会每年节约350美元；

    第二，如果你不采取节能措施，你将会每年浪费350美元。

    第二项宣传语取得的效果远远好于第一条，原因是它是以损失为基本出发点的。


    措辞之所以会有效，是因为人们在做出决策时通常会怀有一种盲目和被动的心态。





    # 如何抵制诱惑


    # 1. 运用理性思维系统

    运用“理性思维系统”计划长期利益时，预测“直觉思维系统”面临的诱惑，以此制订相应的对策。

    【案例】：奥德修斯与海妖塞壬的美妙歌声

在冷静状态时，奥德修斯会命令他的手下用蜡封住自己的耳朵，以便不受赛壬歌声的影响。同时，他还让手下将自
己绑在桅杆上，这样一来，即使他仍然会隐隐约约听到赛壬的歌声，并因此“冲动”起来，他也无法将自己的船划向
海妖，从而避免了诱惑背后的风险。

    # 如何控制社会对我们的影响

    【案例】高压下的群体服毒事件

神父吉姆·琼斯是人民圣殿的创始人和领导人。1978年，为了逃避别人对他逃税的指控，琼斯带领1000名追随者从
旧金山来到了圭亚那的一个地方定居下来，并将那里命名为琼斯镇。

由于面临联邦政府关于其涉嫌虐童的调查，琼斯命令手下的信徒毒死自己的孩子之后再毒死自己，并为他们准备了
好几缸毒药。有几个信徒提出了反对意见，也有人大声抗议，但他们都受到了压制。

迫于琼斯的命令和群体的压力，父母们不声不响地毒死了孩子，然后再毒死自己。他们死的时候躺在一起，手牵着
手。


    # 社会为什么以及如何影响到我们的工作？

    • 大部分人会从别人那里学到自己需要的东西，无论是个人还是社会，都可以从相互学习中得到发展。同时，我们的错误认知同样也来自于别人。当社会影响使人们持有错误认知或偏见时，适当的助推便会产生作用。

    • 助推最为有效（不管是正面效果还是负面效果）的方式之一来自于社会影响。



    # 社会影响主要分为两种：

    • 信息

    如果有许多人都持同一种观点，做同一件事情，那么你也会认为他们的观点和做法也是最适合自己的。


    • 同侪压力

    如果你非常在意别人对你的看法（也许你会错误地感到别人总在注意你），那么你很可能会选择随大流，从而避免别人对你的非议。


    • 措辞
    【案例】：

1. 少女们看到别的少女生孩子，就会增加她们怀孕的概率。

2. 肥胖能够传染。如果你最要好的朋友发福，那么你也有可能变胖。

3. 广播公司之间都会互相模仿，从而通过节目塑造出让人难以理解的新的社会时尚。（比如《美国偶像》这一真
   人秀节目。）


    # 利用从众心理优化人们的行为

    【案例】：政府想鼓励人们多运动，如何做？

1、政府强调一下这个问题；

2、有影响力的公众人物发出号召；


    # 如何避免社会影响

    • 忘掉别人眼中的自己

    其他人对你的关注程度实际上往往低于你自己想象的程度。


    • 发现文化和政治中的从众现象

    1、广告人经常强调“大部分人都喜欢”他们的产品，或者“越来越多的人”已经改用他们的产品。他们总是竭力要告诉你大多数人的做法。

    2、政府或者政党官员的候选人会强调“大多数人”已经转投到他们的阵营。


    • 改善人们的盲从心理

    1、措辞坚决的禁令

    2、用数字助推戒烟戒酒

    3、和邻居争做节能标兵


    • 实现更好的社会引导

    通过询问人们行动的目的可以形成助推，如果进一步询问他们的行动时间和行动方式，则会加大助推。

    【案例】：

1、当研究人员衡量人们的意图时，他们也会影响到人们的行为；

2、当你预测什么东西会流行时，反而会导致什么东西流行；

3、在调查时，研究人员们发现，那些被调查的人倾向于看到自己的行为与别人相似，而不愿意成为特例；




    # 如何提高选择满意度


    # 创造最佳的选择环境

    人们经常遇到的难以抉择的几种情况：

    • 先享受为快

    当选择和选择的结果并不能同时出现时，便会出现自制力的问题。

    一个极端的情况便是后见效行为，比如锻炼、牙线清牙和节食。这些行为首先要投入，但见效很慢，许多人很难长时间地坚持下来。


    • 困难程度

    • 频率

    对于事关重大并且发生概率小的选择，我们可以适当地应用助推。


    • 反馈

    通常情况下，我们只能得到我们选中的选项的反馈，却不能得到我们拒绝的选项的反馈。


    • 为未来做选择

    人们无法判断自己的选择将会如何影响自己的生活时，即使选择再多，人们也很难从中受益。





    # 如何优化选择体系


    # 选择体系六大原则：

    • 动机（iNcentives）

    设计动机选项的一种方法是对某种特定选择体系提出下列4个问题：

    1、谁使用？

    2、谁选择？

    3、谁掏钱？

    4、谁受益？


    • 理解权衡（Understand mappings）

    一个好的选择体系能够使人们改善自己的权衡能力，从而做出对自己有利的选择。

    种做法是使各种选择的信息更容易理解（比如将数字信息转化成更容易实施的内容）。


    • 默认选项（Default）

    人们在面对选择时，内心都会有一个默认选项，那便是不需要选择者付出任何努力的选项。


    • 反馈（Give feedback）

    帮助人们改进其行为的最好方法是提供反馈。设计精妙的反馈系统能够告诉人们的行为是否得当。


    • 预计错误（Expect error）

    一个设计精妙的系统允许它的使用者出错，并对此给予最大限度的宽容。


    • 结构性复合选择（Structure complex choices）

    随着选择范围的扩大，人们越来越倾向于使用简化策略。

    随着可选项目越来越多、越来越复杂，好的选择体系会提供一种结构，而这种结构会影响到最终的结果。





    # 名词解释


    【经济人】：

    即假定人思考和行为都是目标理性的，唯一地试图获得的经济好处就是物质性补偿的最大化


    【社会人】：

    社会人是在社会学中指具有自然和社会双重属性的完整意义上的人，与“经济人”相对。通过社会化，使自然人在适应社会环境、参与社会生活、学习社会规范、履行社会角色的过程中，逐渐认识自我，并获得社会的认可，取得社会成员的资格。


    【诱惑力】：

    比如腰果，我们在“冲动状态”下，比“冷静状态”下吃的更多，我们便称这种东西具备了诱惑力。


    【认知偏差】：

    指人们根据一定表现的现象或虚假的信息而对他人作出判断，从而出现判断失误或判断本身与判断对象的真实情况不相符合


    【集体保守主义】：

    群体倾向于坚持已经形成的共同认知，即便遇到新情况和新要求也会如此。


    【聚光灯效应】：

    刺激反应相容性效应：人们都希望自己所接收到的信号（刺激）能够产生他们希望发生的行为。如果两者出现冲突，人们就会出现错误的行为。
