        # 《远见——如何规划职业生涯3阶段》笔记


        作者：Seiya

        时间：2018年03月14日


        # 职业生涯的三大阶段


        # 第一阶段（工作0-15年）核心目标：寻找甜蜜区

        所谓甜蜜区，指“你热爱的”跟“你擅长的”和“社会需要的”三个区域的交集。

        在寻找甜蜜区过程中，必然会有很多错误的选择和弯路，但在这一阶段要多做尝试，找到自己真正喜欢的东西。如果在第一阶段就选择“钱多活少离家近”的工作，代价就是损失巨大的机会成本，这会让你在余下的职业生涯中毫无资源储备。这一阶段，你需要克服很多心理的恐惧和外界的困难，在不断的尝试和失败中，寻找自己的甜蜜区，并为今后的发展储备充足的职场燃料。


        # 第二阶段（工作15-30年）核心目标：锚定甜蜜区

        所谓“锚定甜蜜区”，就是把更多的精力聚焦在自己的“尖峰点”上，也就是你的天赋和热情所在。

        在这个阶段你需要完成：从“执行者”到“领导者”的角色转换。

        职场中优秀的领导者，一定不是高高在上、一直高空飞行的飞行员，也不是始终插手细节、离不开地面的扫地机，你要做到随时调整高度，收放自如。

        你要不断问自己这三个难题：我擅长什么？我爱好什么？这个世界需要什么？

技能精通的秘方就意愿和时间。我的理解是专注力×时间×改进=精通，当你投入足够专注力和时间，再不断反思如何改进，你就可以精通一个技能。

保持一直向前的动力，就能持续成功。不要放弃自己的理想，实现自己理想的一个窍门是：如果你想做什么事情，就找到那个做得最好的人，然后跟着他们做。

把眼光从小圈子中解放出去。你只有不断的开拓眼界，才能不断的提升自己。

你必须自己帮自己开门。你是最重要的核心，你必须不停地问自己‘另一边有什么，我为什么不能去呢’。永远不要停下对未来的规划，要努力让自己成为幸运儿。


        给初任管理者的小建议：

        （1）时刻注意仪容、态度和举止。这些特征都会给员工的状态造成影响，他们会比过去更加仔细地观察你，并且会根据你的表现来调整自己的表现。你的一举一动都在向员工释放信号，要仔细斟酌会给他们造成的影响。

        （2）着手建立核心小团队。每一个领导都需要一支核心队伍，由那些能够高质量完成任务的亲密同事组成。注意，不要选那些和你相似的人，而要找那些能够增长你的长板和弥补你的短板的人，并考察他们的目标、价值观和关注点是否跟你一致。

        （3）完成心态上的转变。你要把自己定位成一名服务者，你是在帮助你的员工实现自我价值，而不是压榨他们为你创造价值。

        顶级首席执行官需要具备的5大特质:

        （1）诚实和契合。这个主要是指和企业的文化要有契合。

        （2）智力上的好奇和敏捷。顶尖的准首席执行官能够将各种问题综合、联系起来。读书，看报，提问、自我纠错都是其中的途径。

        （3）提升业务业绩的历史记录。你要具备“决心和自控力”。决心是不畏艰险专注于某一目标的能力，自控力指抵抗分心和诱惑的能力，拥有极强的决心和自控力的领导者总是能取得更好的成果。

        （4）真实、自我意识以及平衡。有弱点和不完美都是正常的，但是如果对于成功和失败没有足够的自我反省和反思，那就危险了。

        （5）活力和热情。吸引更多的人才聚集到你的团队里。



        # 第三阶段（工作30-45年）：优化长尾，发挥持续影响力

        职业生涯的第三阶段，一般会在一个人55-70岁，甚至年纪更长的时候。

        这一段的核心发展目标是优化长尾，发挥自己的持续影响力。你可以开始进行从“行动者”到“建议者”的角色转换。从之前挑大梁做决策的人，转换成幕后给建议的智囊团。





        # 三种职场燃料


        # 燃料一：可迁移技能

        可迁移的技能是指在工作过程中，处理问题的基本能力。

        比如，解决问题的能力、沟通和说服的能力、帮助和求助的能力、吸引人才的能力、情商等等。这些技能不仅能够帮助你完成当下的工作，还会是你下一份工作乃至整个职业生涯的窍门砖，因为世界上任何一份工作都是为了解决某个问题而存在的。

        此外“人才吸引力”也是重要的可迁移技能，你能吸引多少优秀的人才，也会决定你的事业会攀升到怎样的高度。


        • ① 解决问题的能力：

          这是企业非常注重的员工能力，很多企业都是以结果为导向的，能够解决问题，可以帮助你抢占公司资源尽快做出成绩，要知道，在任何公司资源都是有限的。速度、简洁、节省、新颖这四个解决实际问题的能力都能让在企业里你大受欢迎。


        • ② 说服式沟通技巧：

          能将自己的观点以清晰、简洁的方式呈现出来都是一种基本技能。它包含语言的表达、沟通说服能力和书面说服能力。说服能力是让人心悦诚服的接受你的观点，要提供强大的证据链、严谨的逻辑、系统的大局观以及对方重点关注的利益点，切忌不能强行灌输你的观点，要注重方法和技巧。


        • ③ 完成任务的能力：

          这是指执行能力，《致加西亚的信》这本畅销书就是强调执行能力，尤其是注重工作的主动性和创造性。在工作的前期，你还不能独当一面，执行能力强的员工很受部门领导的欢迎。


        • ④ “人才引力”：

          这是指吸引人才的能力，这是一个高级能力，比如大家熟知的马云、雷军都具备很强的个人魅力，有能力吸引和调动尖端人才的个人领袖通常都能成功。亚当·格兰特在《沃顿商学院最受欢迎的成功课》中强调：最成功的付出者就是付出超过获取的人，跻身最杰出和最幸福行列的机会会比别人大得多。但是你要注意不要成为企业的老好人，付出尺度的把握很重要。


        • ⑤ 情商：

          这是指你对自己和他人情绪的识别和理解能力，以及用这种意识管理你的行为、处理人际关系的能力。现在大家都明白智商是一系列测试得出的数据评分而已，就像大学的考试试卷，分数高并不代表你的能力强。情商和逆商在人生中其实占据更加重要的地位。关于情商我会在后期写一个书评，这里就不详细介绍了。


        • ⑥ 搜商：

          这是现在出现的一个新技能，你要能在大量的信息中找到你需要的信息，要知道如何区分可信和虚假的信息来源，也要知道该如何用清晰、有说服力的方式呈现数据，最核心的是要知道如何核实搜索结果。



        # 燃料二：有意义的经验

        有意义的经验，是你职业生涯生态系统的重要组成部分。

        历史经验的多样性会让你成为一个复合型人才，并且能够帮助你构建一个更加稳健的职业生涯。

        丰富工作经验的多样性可以从很多维度进行，比如创业、跨国工作、做志愿者、去留学等等。你需要在不同环境中尝试不同的事情、试验不同的做事方法，这样才能把自己塑造成一个更强的“职场混血”。

        ① 分别到一个大公司和创业公司的环境中工作；

② 找机会去国外工作或去几个不同的城市工作；

③ 发起一些新项目、处理一次危机，并在明显存在个人失败风险的情况下筹办一场大型活动或演出。


        # 燃料三：持久的关系

        职场中，每个人的周围都会出现许多关键人物，并影响着你的职业轨迹。这些人会为你组成一个金字塔结构的生态系统，它一共分成5层，从下到上分别是：联系人、专家团、关键同事、支持者以及你自己。

        （1）金字塔最底层：“联系人”。“联系人”是价值含量最低的一种职场关系，彼此为对方产生的影响都非常小，甚至可以忽略不计。这种没有深层连接，仅仅互留联系方式的关系称作“未经改良的职场燃料”。

        （2）金字塔第二层：“专家团”。“专家团”是比联系人稍有价值的职场关系，他们是拥有特殊的知识和权力、能帮助你取得成功的人，是要你去主动结识好培养的。不过培养这种关系最好的方法不是求助，而是主动为他人提供帮助。

        （3）金字塔第三层：“关键同事”。“关键同事”是指目前对你产生决定性影响力的5～10个人，排在榜首的是你的上司，他们对你的升职加薪有直接的影响。此外，某些同事和下属也可以成为关键同事圈的一员。

        （4）金字塔第四层：“支持者”。“支持者”是为你提建议、拥护你、激励你的导师和帮手，人数一般不会超过5个。他们是你在职场中前进的推手，会让你的职业生涯出现非同一般的拐点。

        （5）金字塔第五层：你自己。任何职业生态系统的核心都是你自己。你就是你自己的品牌经理，你需要期盘点自己的职业生态系统，确保它积极活跃地发展，并为你提供的是积极正向的支持。



        # 职场燃料清单


        # 1号燃料的盘点清单:

        ① 学位、专业证书；

        ② 语言，包括音乐和计算机语言；

        ③ 上司和同事在你的360度绩效评估中经常提到的优点；

        ④ 你收到的关于情商的评价；

        ⑤ 你的“人才账户”。



        # 2号燃料的盘点清单

        ①个人旅行；

        ②国外工作经历；

        ③企业管理或创业经验：

        ④社区或志愿者活动：

        ⑤包含你个人贡献的大型活动、产品发布或知名的项目：

        ⑥公开演讲、写作或表演的经验：

        ⑦教学、咨询或指导的经验：

        ⑧兴趣爱好、业余活动和工作之余的热情所在：

        ⑨其他生活中的经验和挑战；



        # 3号燃料的盘点清单

        • ① 联系人：

          包括我们的领英联系人、邮箱联系人、校友、同事等。人际关系的构建者会首先尝试帮助别人，他们不会有所保留。但是注意要提供的是有工作价值和情绪价值的帮助，而且最好是在你力所能及的情况下，控制好度，不要成为老好人。


        • ② 专家团：

          职业生态系统中这是一个较高的层次。这些专家是拥有特殊的知识和权力、能帮助你取得工作和职业生涯成功的人。你要理清你的专家团中都有谁？它覆盖了哪些领域，你在哪些方面又需要更多的帮助呢？你是否用积极的付出回应了他们的请求，做好了你分内的事呢？


        • ③ 关键同事：

          是在目前的公司里对你的发展拥有决定性影响力的5～10个人，排在榜首的是你的上司。


        • ④ 支持者：

          是在职业生涯中帮忙提建议、拥护你、激励你的导师和帮手。他们通常为数不多，大概不到5个人。人生导师分为三类：

            ⑴ 明星：他们是成功的行为榜样，告诉我们如何成功；

  ⑵ 贤者：他们就像苏格拉底，并不为我们提供答案，而是教我们如何思考；

  ⑶ 策动者：他们激励我们，鞭策我们，偶尔迫使我们踏出关键的一步。

          在找到了能推动你职业生涯前进的人时，你就要想尽办法帮助他们、为其所用，或者花时间与他们相处。重要的是，要感恩！你要让他们看到他们的影响力，尽早和尽可能地感谢他们。


        ⑤你自己：任何职业生态系统的核心都是你自己，你只有自己努力，才有可能成就你自己。



        # 如何善用你的时间资产

        如果你问一个资深的财务顾问，如何让你的投资获得最高的长期产出，他会告诉你，关键是“资产配置”。职业生涯也是一样的道理，只不过关键的变量是你如何投资时间，因为时间是你人生中最具价值的个人资产。


        # 1. 精进一项技能至少需要 3-5 年的时间

        每个人的生涯目标不同，对应的时间投资方案也肯定不同。不过，无论你的目标是什么，都至少要建立一个意识：没有一蹴而就的事情，想要在特定领域里取得一定的成就，至少需要1万个小时，也差不多就是3-5年的时间。



        # 2. 合理利用碎片化时间

        聪明的时间投资者会把时间分成大块和碎片时间。利用大块的时间来精进最重要的事情，利用碎片化时间来处理其他任务。

        一个高效而健康的时间分配，应该要涉及工作、家庭、锻炼、社交、学习和休闲等项目。不同的职业生涯阶段，各个项目的分配比例也会有很大的不同。



        # 3. 建立时间投资档案

        每年至少盘点一次时间档案，通过审视自己时间投资的实际情况，你就能知道自己把重点放在了哪里，以及哪些方面正在为你追求的结果让步。

        时间档案会告诉你可以在哪些方面做出取舍。

        著名畅销书《跃迁》中古典介绍了生命之花八个部分的时间规划，更为具体，你也可以做参考：

1、职业发展；

2、财务状况；

3、个人成长；

4、自我实现；

5、家庭；

6、健康；

7、朋友与重要他人；

8、娱乐休闲。

        《跃迁》中推荐了生命之花顺序是：

        ① 先安排硬时间：职业、财务

        ② 其次安排家庭时间和健康时间；

        ③ 安排个人成长和自我实现时间；

        ④ 娱乐休闲和重要他人安排最后，因为你有动力抽时间完成。



        # 五、职场路径设计

        在尝试建立新的职场路径或者在多个选项中抉择时，运用“职场路径向导”回答三个问题：

        1.你的职业理想是什么，或者至少假设一个你可能想要达到的目标。

2.你目前手上有什么职场燃料？

3.你需要什么职场燃料才能实现这个终极理想？思考它能否：

    ①增加你目前不具备的可迁移技能？

    ②帮助你重塑自我，改变职业生涯的方向？

    ③建立新的人际关系，并拓展职业生态系统？

    ④获得你目前没有的重要证书？

    ⑤加速你的探索步伐，即通过实践验证自己真正擅长和热爱的东西？

        结合职业燃料的清单，你可以对照一下看看那些地方不足，如果你觉得你需要弥补的燃料太多，你也不必感到失落和绝望，职场燃料清单只是给你一个参考，很少有人能够全部做到，在目前你只需要做到一部分就可以了。

        给自己设定一个清单，一年达到什么目标？每个月设定一个清单——做什么具体行动来实现你的目标？每天设定一个清单来落实。清单一定要把握三个原则：简单、可测和高效。有兴趣的朋友可以看看阿图·葛文德的《清单革命》，它会帮助你如何设置清单。

        我前面介绍过《微习惯》说过：大量的目标，繁重的目标会把你压垮，所以你设置目标一定要小一点，一次一个。不要给自己压力，这样达到目标你会有满足感，超出了目标你会有成就感。

        最后，记得月目标达成后，要给自己一点小奖励，可以是大餐、郊游、喜欢的衣服、饰品等等，来帮助自己巩固习惯。



        # 六、职场四大黄金问题


        1. 我是否正在学习和成长？

        2. 我是否正在对某些人、现在的公司，乃至整个社会拥有影响力？

        3. 我体验到乐趣了吗？

        4. 我是否得到了适当的奖励，并创造了经济价值？

        这四个问题贯穿于职场三大阶段，是你在漫长职场生涯中需要不断的提问、回答和反思，每年的年末问一次自己，可以让你深入的思考下一年应该怎么做，如何让自己工作的更有意义，更快乐、让自己不断的进步，而不是逐渐的走向衰落。



        # 七、摘录


        1. 职业生涯不是短跑比赛，而是一场至少长达45年的马拉松。

        2. 职场中优秀的领导者，一定不是高高在上、一直高空飞行的飞行员，也不是始终插手细节、离不开地面的扫地机，你要做到随时调整高度，收放自如。

        3. 世界上任何一份工作都是为了解决某个问题而存在的。

        4. 在不同环境中尝试不同的事情、试验不同的做事方法，这样才能把自己塑造成一个更强的“职场混血”。
