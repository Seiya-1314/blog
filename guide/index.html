# 项目经历

# 法家云后台管理系统（web 应用） 2019.05 - 至今

【项目描述】

利用新的前端技术栈：Vue、webpack、element-ui 等，重构后台管理系统。

【我的工作】

我在此项目中单独完成了后台管理系统的前端开发工作，在这个项目中，主要难点在于用户权限管理，前端路由管理， 全局数据管理。这里我参考了(https://github.com/PanJiaChen/vue-element-admin)这个项目中提供的成熟方案，进行了组件的开发。同时也参考了路由管理，全局数据管理的方案。对于用户权 限管理，我这里使用了(https://github.com/vueComponent/ant-design-vue-pro)提供的方案，在路由守卫中定义各种权限的跳转逻辑，在每次路由发生变化时，根据当前的权限，跳转到不同的页面。


# 法小弟（微信小程序）2019.03 - 至今

【项目描述】

利用整合的律师资源，对接微信小程序，利用社会化网络进行产品推广，拓展业务。同时，提高产品便捷性、可用性， 提高用户体验。

【我的工作】

我在此项目中单独完成了前端微信小程序开发，这个项目中，进行了前后端分离开发。主要难点在于需要重新熟悉小程 序的相关技术栈。在这个项目中，我熟悉了分享后登录拦截的实现方案、本地缓存策略的实现、数据埋点方案、小程序 版本升级方案、特殊机型适配方案等。


# 法家云（移动端web应用）2018.02 - 至今

【项目描述】

向支付宝用户提供相关法律服务。凭支付宝流量优势，整合全国律师资源，利用互联网技术，提高行业效率，建立全新 的律师评价体系，打造一个法律电商平台。

【我的工作】

接手上任同事的工作，独立完成支付宝平台web应用的后续需求开发，主要技术栈为jQuery。这个项目中，我遇到的 主要难点是性能优化的问题。由于首屏数据量过大，采用的本地缓存的策略，用户每次进入首页，先判断缓存是否过期 ，如未过期则加载本地缓存，反之重新获取首页数据。


# 口袋故事（微信小程序）

【项目描述】

口袋故事是一个儿童故事平台，为 0-12 岁的孩子提供精选音频播放内容。

【我的工作】

我在此项目中独立负责微信小程序的前端开发工作，这个项目中，主要难点是音频播放进度条的实现。这 里使用了微信小程序自带的 <slider> 组件进行了封装，同时通过相关事件对播放进度进行了控制。同时，为了突破小程序客户端的容量限制，使用了微信官 方提供的分包方案。
