        # 计算机网络体系结构


        作者：Seiya

        时间：2019年09月02日


        # 介绍


        OSI 体系结构：概念清楚，理念完整，但太过于复杂，不实用。

        TCP / IP 体系结构：含一系列构成互联网基础的网络协议，是 Internet 的核心协议，被广泛应用于局域网和广域网；

        五层体系结构：融合了OSI 与 TCP / IP的体系结构，目的是为了学习和讲解计算机原理；



        tips

        低三层为通信子网，负责数据传输 高三层为资源子网，相当于计算机系统，完成数据处理； 传输层承上启下
