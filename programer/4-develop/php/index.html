      # PHP学习

      我为什么学习PHP？其实在开始学习PHP的这个时间段，我已经在断断续续的学python有段时间了，但由于公司需要快速开发一个后台管理系统，所以还是选择了PHP这个方向。主要也是考虑到这个时候PHP的学习，既可以让自己加入到这个项目中来，也可以了解后端开发的一些知识，还同时为python后端开发打下基础。


      整篇博客是记录自己在学习PHP过程中的一些资料和总结，部分是网上找到的整理好的内容，在这里只是做一个资源的汇总，方便自己能够更好的复习，希望能够达成这个目标。
