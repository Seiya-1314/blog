        # 社会成本问题——谁用得好就归谁


        作者：Seiya

        时间：2017年12月26日


        科斯定律的一个重要含义：谁用的好久应该归谁。本质上就是人们对稀缺资源的争夺。

        举几个例子：

1、淘宝网店数据归谁所有？你可以说：数据是用户产生的，归用户所有。商铺可以说，数据是在他们店里产生的，
归店铺所有。淘宝平台也可说，数据是平台产生的，归淘宝平台所有。你觉得数据应该归谁，根据科斯定律，数据
应该归淘宝平台所有，因为淘宝平台能用好这个数据。

2、Kindle 阅读器中的数据产权，归谁所有？我们在使用 Kindle 阅读器阅读的过程中，所标注的笔记，这些笔
记的数据信息归谁所有？你可以说：数据我用好标注的，归用户所有。但是，由于你是在书中做笔记，所以这个笔
记数据应该归作者或出版社所有。亚马逊也会说，这些数据是在平台产生的，归平台所有。根据科斯定律，数据应
该归亚马逊所有，因为它们能用好这个数据，我使用 Kindle看一本书时，可以看到别人在此位置做的笔记。


        # 隐私权与公共安全的权衡


        今天大数据的使用，经常引发隐私权应该归谁的问题。好多人对隐私权非常敏感，觉得隐私权一点都不能出让。当然我能够理解这种感受。

        我今晚跟谁在一起，跟谁吃饭了？跟谁坐飞机出去旅游了？这些数据对别人可有可无，但对我来说可能非常重要，所以保护隐私确实挺重要的。

        但是有些数据，对大众来说也很重要。比方说，一些犯罪分子的数据，公安机关拿到这些数据，它就能够产生极大的效用，避免很大的损失。这时候公安机关应不应该有一点权力，去征用私人的隐私权？


        根据科斯定律的预见，关于隐私权的争论，最后解决的方案不会一边倒。

        既不会允许政府肆无忌惮地侵犯个人隐私权，个人也不可能绝对地保有所有的隐私权。最后一定会在这两者当中取得一个平衡，而当中的一个指导原则，还是科斯定律：谁用得更好就归谁。

        再举几个例子：

1、钻石归矿工所有还是白富美？
钻石最早是归矿工的，因为是矿工挖出来的，但你有没有见过，那些矿工脖子上挂满钻石？钻石都跑到哪里去了
呢？没错，钻石不远万里跑到了白富美的身上。

2、女朋友与科斯定律。
只要交易费用足够低，那么不管这个女孩现在跟谁谈朋友，她最后会跟最合适她的人在一起。


        # 小结


        一项有价值的资源，不管从一开始它的产权谁属，最后这项资源都会流动到最善于利用它、能最大化利用其价值的人手里去。

        这是科斯定律的一个重要含义。而在制度设计中，我们应该尽量让这种资源的流动和分配更方便容易，从而提高各项经济资源的使用效率。



        # 课后思考


        问：举一个身边的例子

        # 我的思考：

        投资领域也是如此，谁能用的好资本，谁就能拿到更高的估值，更多的融资额。

        # 同学的回答：

        1、当你努力成为使用某种资源效率最高的人，创造最多价值的人，那么资源会自然逐渐向你靠拢。

        2、设计制度时，让流动和分配成本更低，更利于资源的使用效率。

        # 薛兆丰老师的回答：

        准确理解科斯定律的5个要点（前提是“只要交易费用够低”）：

        1、资源归谁所有，资源用在什么用途上，它跟产权的所有者无关（牛和小麦的故事，最后牛该不该吃小麦，这是跟谁能够产生最大的效益来决定的，跟所有者没关系）。

        2、与当事人的数目也没有关系（咱们数的不是人头，而是经济效益）。

        3、与补偿和不补偿也没有关系（也许补偿，也许不补偿，但这些不重要，重要的是资源最后归谁。火星与亚麻的例子，自从这个案子之后，人们开始认识到，责任应该归于农夫，因为铁路公司付出的社会成本太高）。

        4、随着默认责任的改变，原来补偿的，后来可能就变得不要补偿了。

        5、责任跟收入大不大没有因果关系（就像火星和亚麻的例子，责任跟农夫的收入低，铁路公司的收入高无关）。
