            # 从成本角度理解盈利与亏损


            作者：Seiya

            时间：2017年12月14日


            无论是盈利还是亏损，在经济学的概念里，都是意外发生的。经济学是容纳不了盈利和亏损这两个概念的，我们都把它当作意外来处理。



            # 凡是盈利都是意外


            成本是放弃了的最大代价，放弃的东西越多，代价就越高；放弃的东西越少，代价就越少。如果没有放弃，就没有成本。

            举个例子：

1、我们到邮局花一块钱买一张邮票，假设以后，我们随时可以拿到邮局换回钱，并且换取的成本忽略不计，那么我
们购买邮票的成本其实为 0。因为你什么都没放弃，你随时都可以把钱换回来；

2、我们到汉堡店买汉堡，然后吃下第一口。这一口的成本几乎就是整个汉堡的成本，因为吃下第一口后，这个汉堡
就卖不出去了。这一口就把拿汉堡换回钱的机会消灭了。

3、我们买一辆新车回来，先开半年，这半年的成本是高还是低呢？这半年的使用成本就是新车价格减去此时卖出的
二手车价格，这个成本是很高的。但是再多用半年，这辆车的使用成本反而降下来了。

4、再举一个例子，我再街边花10块钱买个碗，回来后发现这个碗其实是个古董，它价值10W 元，如果我用这个碗
吃饭，那么成本是多少呢？它的成本是10W 元，而不是购买所花费的10块钱。因为放弃的最大代价是将碗换回10W
元的机会。

            也就是说，当我们得知这碗不是一只普通的碗，而是一个珍贵古董的那一刹那，我们获得了盈利。凡是盈利都是意外，英文叫做windfall profit，可以译成“横财”。横财是你意识到这个商品的价值跟你预期不一样的那一刹那发生的。

            盈利提高了资源未来使用的成本。



            # 亏损会降低资源使用成本


            举个例子：

我到古董店花10W 元买了个古董，拿回家发现是假的，它实际价值10元。那么我拿这个玩吃饭，它的成本是10元，
而不是购买时的10W 元。

            我蒙受亏损，只不过是一刹那的事情。我重新调整了对这个碗的预期。在蒙受亏损的这一刻，亏损是意外的，是没有想象到、预料到的，英文叫windfall loss，中文可以译成“横祸”。


            一旦蒙受损失，我们就会重新调整对这个资源的估值，继续使用这个资源的最大代价，就以这个资源现有的价值水平来做估算。

            一旦出现盈利，资源使用成本就会提高；一点出现亏损，资源使用成本就会下降。



            # 重新估值不难，难的是找出盈利点或亏损原因

            在现实生活中，困难的地方，在于你要找到那些给你带来盈利或者带来亏损的资源。因为现实生活中，所有的资源都是组合在一起用的。

            举个例子：

咖啡厅赚钱了，那么它赚钱的原因有哪些？

原因一：服务员态度特别好，那么服务员的服务素质这一块就要升值，因此成本就要增加。因为如果不付出更高的
工资，就留不下服务员；
原因二：选址选对了，那么选址团队的才能就要被重新评估，对他们的选址付出更高的价格；
原因三：咖啡豆质量很好，那么就要付出更多资金，购买市场上，质量更好的咖啡豆；
原因四：连锁店效应，那么我们就要更大的资本，更多的店面提升品牌效应；

            当然反过来也是一样的。如果咖啡厅蒙受了亏损，你得把造成亏损的原因找出来，降低对它们的估值，减少对它们的付费。



            # 小结


            从成本的角度定义盈利和亏损这两个常见的经济学概念：无论是盈利还是亏损，都是意外，都是刹那间完成的。

            一旦出现盈利或亏损，我们就重新评估资源。资源未来使用的成本，就在新的评估水平上进行计算。



            # 课后思考


            问：好生意如果旱涝保收的话，当人们逐渐了解之后，好生意就不再是好生意了

            # 薛兆丰老师的回答：

            这句话有几层含义：

            第一层含义：我们是讲在企业的内部的。在企业的内部，一门生意如果是赚了钱，企业的主管人就要在这企业的各种要素里面，寻找那些能够带来额外收益的、也就是盈利的要素，把它们确定下来。一旦确定下来后，它们的要价就会上涨上去。这些要素就会受到其他用途或者其他企业的争抢，迫于竞争压力，企业所有者也好，要素拥有者也好，他们就会重新评估他们的价值。

            第二层含义：是在企业的外部，一旦出现旱涝保收的生意，其他竞争者就会争相提供替代品。然后他们会夺走原来被你独占的那些用户，被你独占的那些需求。这是第二个含义。

            第三层含义：是那些铁饭碗、金饭碗，一旦人们发现这份工作是个铁饭碗，这份工作是个金饭碗，那么对这些工作的竞争就会加剧，进入的门槛就会逐步抬高，高到后来再进去的人已经得不到什么额外的好处了。
