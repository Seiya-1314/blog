        # 违约与赔偿


        作者：Seiya

        时间：2018年09月05日


        经济学把一份合约、一个承诺看作是一项资产，把违约看作是对资产的侵害。既然对资产有侵害，就有不同的解决方式。


        # 原则一：财产原则


        在英美的合同法里面，按照财产原则来保护的合约称为“具体履行”（答应什么就给什么）。


        举例一：金宝汤公司诉温氏农村案

【背景】：
著名的做罐头汤的公司“金宝汤公司”，跟温氏农场早早就签订了合约，要买他们的胡萝卜。合约规定，胡萝卜收购
价是每吨20多美元。但是那一年胡萝卜特别紧俏，价格涨到了90美元一吨，温氏农场就偷偷把他自己生产的胡萝卜
卖给了别人。

金宝汤公司知道了就起诉温氏农场，要求法庭颁布禁制令，禁止温氏农场把他们自己的胡萝卜卖给别人。

【判决结果】：
判决非常清楚明，合约里面规定温氏农场要给什么，他们就得给什么，具体履行（Specific performance ）。


        举例二：

【背景】：
有个人承包了一家餐厅，餐厅的主人答应承包者说，“如果持续承包五年，合同期满的时候，就会把自家做肉酱的秘
方卖给你。”结果五年合约期到了，这秘方的主人后悔了，不肯卖了。承包者就把这秘方的主人告上了法庭。

【判决结果】：
法庭的判决也很清楚，答应什么就要给人家什么。如果你觉得这秘方价值实在太高，真的舍不得卖，那也可以，你
就可以向这位承包者重新把它买回来。


        # 原则二：责任原则


        有时候不具体履行合同才是有效的（有效违约原则）。


        举例三：

【背景】：
被告有一大批的铁矿砂，要求原告帮他打包，被告就跟原告说这笔可是个大生意，他要接这个单得更新他们的设
备，他们得买一套打包的设备。结果原告就真的听了被告的话，引进了一部新的设备，然后就开始给被告做铁矿砂
打包的业务。

当这笔生意进行到一半左右的时候，市场的情况发生了改变，原告指望的那笔大生意落空了。

【判决结果】：
原告得到了他指望的利润，但是生产用不着真的继续下去，可以避免的损失避免了，这就是有效违约的概念。


        举例四：

【背景】：
有一家人请了一家建筑公司，给他们后院修一个游泳池，合同规定游泳池最深的地方应该有七尺六寸深，也就三米
左右，他们的目的是要跳水。结果这个游泳池修好了以后，最深的也不过是六尺九寸，而在人们本来要跳水那个位
置只有六尺深，跳不了水了，建筑公司违约了。

【矛盾点】：
如果具体履行，就要把游泳池给拆了重新修，这会造成重大的浪费。

【判决结果】：
法官说他们成立一个小组去调查一下，那些有比较深的游泳池的房子值多少钱，那些只有比较浅的游泳池的房子值
多少钱，算出两者当中的差价，这就是违约造成的差价，建筑公司就赔这个差价就可以了。


        # 课后思考


        问：在中国绝大部分的商店买东西，买来的商品都是不能退货的，但是如果你真的买了一件你不需要的商品，留着也是浪费，那你用什么办法来处理呢？

        # 我的思考：

        当二手卖出去，或者送给需要的朋友。

        # 薛兆丰老师的回答：

        第一个卖出去的对象就是原来的卖主，因为他是把东西卖给你的人，他是最了解这产品的性能、价格和来路的，信息不对称的情况最不严重。
