    # 何谓短缺和过剩


    作者：Seiya

    时间：2018年03月27日


    # 稀缺、短缺和过剩之间的区别


    • 稀缺是永恒的，而短缺和过剩都是价格受到人为干预的结果。

    • 当价格过低的时候，人们不得不展开价格以外的竞争方式，来争夺他们需要的商品，这时候就会出现短缺。

    • 而当价格被人为拔高以后，卖家就不得不采用价格以外的方式，才能把东西卖出去，这时候就会出现所谓的过剩现象。


    # 稀缺

    如果一种商品，用出价的高低来竞争，而且只用出价的高低来竞争的话，这种商品就永远不会短缺，而只会稀缺。

    【案例】：

1、奔驰的汽车是稀缺品；

2、蒂芙尼的钻石也是稀缺品；

3、五星级酒店里面的总统套房也是稀缺品；

    # 短缺

    短缺不是供应的减少，而是商品的价格受到抑制，人们不仅仅要通过出价的方式去竞争，还得依靠其他非价格的方式去竞争的一种特殊的现象。

    【案例】：

1、出租车短缺，你需要站在那里，等上20分钟、30分钟才能打倒车，你得付出时间上的竞争才能打到车，这是短
缺；

2、春节期间买不到火车票，你得找上黄牛党才能买到火车票，这是短缺；

3、过去计划经济年代，你有钱买不到吃的，买不到穿的，买不到用的。你得有粮票、你得有布票，你得有电灯票，
才能买到东西，这是短缺；

    【思考】：从经济学的角度看，为什么那些根据排队来分配的国家很穷

    人们很忙，每天为了争取到一点物资，都要花很多的时间，但整个社会是低效的，因为排队所耗费的资源不能让社会其他人得益。

    比方说苏联搞计划经济的时代，人们就非常习惯排队。那个时候，每个人上街口袋里面都会放一个网兜，一看到有人排队，你就加入他们的队伍，至于能获得什么不要管。时间不值钱，排队获得的商品才值钱，这也被称为“短缺经济”。


    # 过剩

    过剩不是供给过多，而是产品的价格受到人为哄抬，以至于卖家不能简单地通过价格，就能够把产品卖出去。

    【案例】：

1、小时候听说资本主义腐朽，牛奶都要倒掉，肉要腐烂掉，土豆烂在地里，当时的解释是说资本主义是盲目的，没
有计划性的。其实那是过剩，农产品的价格被人为拔高了，以至于农民必须展开其他方式的竞争；

2、我们都说农民需要扶持，怎么扶持呢？政府把农产品的价格拔高，而由于农产品价格高于市价，农民多生产就多
赚，当然他们就会拼命地生产。价格过高，你就不能顺利的卖出去。而且政府的资金是有限的，农民和农民之间就
会展开对政府配额的竞争，看谁得到扶持；

    【思考】：过剩有什么解决办法？

    1、政府减少补贴，过剩就会消失；

    2、政府必须限制农民的产量，这时候农民之间，就会争夺所谓的配额；

    3、强制勒令那些超出需求量的产品，让他们自找门路进行处理。



    # 课后思考


    问：举一个短缺或过剩的例子，并提出解决方案

    # 我的回答

    1、滴滴春节期间就有短缺现象，通过提高价格的方式，解决短缺的问题；

    2、淘宝上经常有商品出现过剩现象，需要通过其他促销方式进行售卖；

    # 同学的回答

    共享单车初期出现短缺现象，需要预约和抢才能骑到；商家大规模投放后，短缺现象变成了过剩现象，需要投入其他的成本去运营，去调度，政府甚至出台相关政策限制单车投放数量。

    # 薛兆丰老师的回答

    【案例】：1971年到1980年，美国持续了近十年的石油短缺现象是如何一夜之间消失的？

    背景：

    在1971年8月15日，当时的尼克松总统发表全国电视讲话，他说：“我今天下命令冻结整个美国的所有工资和所有的价格。”而从那天以后，石油产品的价格就被他摁住了。

    发展：

    1、最开始的时候，人们不得不排长队，在加油站门口等着加油；

    2、后来，人们觉得排队太浪费时间了，为了追求损失最小化，结果就有了“单双号”的规定；

    3、但是这种方法也无法解决问题，于是出现了“预约加油”的规定；

    4、有些加油站就推出了“熟客优先加油”的规定；有些加油站推出了“捆绑销售”，通过捆绑洗车、保养服务挽回损失；

    5、有些地方出现了“星期天不准开车”的规定；有些地方出现了“一家最好就开一辆车”规定；

    6、最后国家还出动了电视媒体，要求大家节省能源，不要天天洗澡；为了节省能源，甚至高速公路上的路灯也没有了；

    结果：

    但是这些都不能够解决问题，直到有一天，里根总统上台，他上台的同一个星期签署了一份命令，“从现在开始，我要解除一切针对石油产品的价格管制”。一夜之间，油价上涨到了恰如其分的位置，人们开始节省使用各种各样的石油产品。

    更重要的是，有一些长期没有开工的油田开工了。为什么？因为那些油田的生产效率可能不是那么高，地底下的油可能比较深，要把油从地底下抽出来成本比较高，在价格管制下，那个价格得不偿失，所以他们停工了。这时候，由于石油的价格已经上涨到恰如其分的价位，这些油田重新开工了，供应增加了。
