      # 各种竞争规则孰优孰劣


      作者：Seiya

      时间：2018年03月26日


      # 关于各类竞争规则

      • 在所有的竞争规则当中，没有任何一个规则是比别的规则更公平的，不同的人站在不同的立场就有不同的偏好，不同的选择

        选择不同的规则，那么禀赋不同的人，胜出的机会就不一样

        • 暴力：身强力壮的人、喜欢练武的人竞争力更大；

        • 智力：天生智商高的人、有文凭的人竞争力更大；

        • 身份：根正苗红的人、有社会地位的人竞争力更大；

        • 劳动时间：勤勤肯肯的人竞争力更大；

        • 先到先得：时间多的人、平时百无聊赖的人竞争力更大；

        • 社会关系：善于交际的人竞争力更大；


      • 不论你选择哪一种竞争方式，人们都会按照这种竞争方式去竞争，而竞争本身就会耗费资源

        一旦把竞争规则确定下来，人们就会朝着一个特定的方向去努力。这个努力的过程，就会消耗真实的资源，时间、金钱、注意力等等。

        * 暴力：人们会经常去练武，练武本身就是一种成本；

* 智力：人们会整天学习，整天考试，这也是成本；

* 身份：按职位高低，那么就有攀附权贵的成本；

* 先到先得：会浪费许多的时间成本；

* 按需分配：根据社会亏欠你多少来分配，会浪费许多装穷的成本；

* 随机分配：摇号分配的话，那么参与摇号的人会越来越多；落到那些其实并不需要的人手中，这也是成本；

      • 在竞争过程当中所损耗的资源，通常都是一种无谓的损失。也就是说，对竞争者本身，对那些赢家来说，这是值得付出的代价，但是对社会其他人来说，他们从中得不到好处；

        所有这些竞争带来的成本，其他人都是很难得益的。

        • 暴力： 如果你练武很出色，最后抢到了你需要的东西，但社会上的其他人，却不能从你身上多出来的肌肉得到好处；
        • 智力： 你得到过多的教育，别人也不可能从你身上得到什么好处，因为今天又互联网，我可以自己搜索答案；
        • 身份： 有些人攀附权贵，你不可能从他身上得到其他的好处；
        • 先到先得： 你花很长时间排队得到想要的东西，你觉得很值得，但社会上其他人，却不能从你排队浪费的时间中得到任何好处；

        所有这些努力本身，对于得到资源的人来说似乎都是值得的，但是对于社会其他人来说，所有这些努力都是一种无谓的损失。


      • 用出价的办法来竞争的话，有一个特点，跟其他的竞争方式不一样，那就是每当人们为了赚更多的钱，从而在出价的过程当中胜出的时候，人们不得不向社会其他人提供他们所需要的产品。这时候，整个社会因为竞争带来的不是无谓的损失，而是更多的财富。

        这种竞争方式，会指引人们去从事一些积极的、有收益的、对社会其他人带来福利的服务。

        它带来的不是多余的肌肉，不是多余的教育，不是多余的社会关系，也不是多余浪费的时间，它带来的是，社会其他人也愿意付钱购买的商品或者服务。

        在竞争的过程中，它不会因为竞争的过程消耗资源，使得整个社会的饼越来越小，而是在竞争的过程中，整个社会的饼，整个社会的福利，是越做越大的。

        从公平的角度看，出价高低不是一个更公平的办法。但是从效率的角度，从社会财富积累的角度，从如何减少竞争带来的无谓损失的角度来看，出价高低是比别的竞争方式更有效率的，它直接带来了社会财富的高速集聚。


      【思考】：中国经济改革是什么？改革开放30多年带来的巨大成果是什么？

      中国经济改革，其实是一种竞争规则的改变。

      过去，我们看你的排队时间，看你的出生地点，看你的政治面目，看你的需求大小，变成了另外一个规则，也就是“出价高低”！

      今天，我们所需的绝大多数商品服务，基本都是按照出价的高低来分配的。

      就这么一个简单的社会竞争规则的转变，造成了整个社会财富积累的巨大变化。



      # 课后思考


      问：有什么商品是不应该按出价高低来分配的？钱不能买到什么？

      # 我的回答：

      1、稀缺医疗资源，比如“捐赠器官”等；

      2、稀缺优质教育资源，必须要“学区房”等方式才能拿到资格；

      # 同学的回答：

      大学会录取最优秀的学生，而不是那些付钱最多的学生；医院急诊室是根据病情的紧迫程度，而不是是否支付额外费用对待病人；

      # 薛兆丰老师的回答：

      卖家到底需不需要挑客，也就是说卖家有没有必要选择把东西卖给谁？

      • 如果有必要的话，那么他卖的东西就不能仅仅是价高者得。与此同时，由于他挑客了，他选择了他的顾客，这时候他的顾客的范围就缩小了。这些更小范围里面的顾客，他们愿意出的价格就会降低，卖家就得承受因为他挑客而蒙受的金钱上的损失。如果他觉得这么做是值得的，那么他就会这么做。

      • 倒过来，如果卖家并没有兴趣去挑客，也就是说卖给谁都无所谓，只要能卖出去多赚钱就可以了的话，那么他们就应该采取价高者得的办法。

      每当我们指责以金钱为标准的时候，我们要牢牢记住放弃了金钱的标准，我们还得选另外一个标准。我们得看，另外一个标准是不是比用金钱的标准更好。
