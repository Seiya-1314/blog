    # 易耗品和耐用品


    作者：Seiya

    时间：2018年06月06日


    # 欧文·费雪的《利息理论》


    收入是一连串的事件（Income is a series of events ）

    未来发生的一件一件一件的事情，连续加起来，就是你的收入。这是一种很新的角度，看所有资本是怎么提供服务的角度。



    # 易耗品的定义

    一次性就能消费完毕的商品。


    # 耐用品的定义

    商品提供的服务、发挥的功能，必须是在一个时间段内慢慢发散出来，需要你有时间才能享受到。


    举例一：生活中的易耗品和耐用品

* 易耗品：

  - 一个苹果、一个鸡蛋、一杯牛奶、一份奶酪（奶酪是经过一段时间生产的，属于未来的易耗品）

* 耐用品：

  - 一幅油画（需要搭上时间，才能欣赏它）、一颗苹果树（需要在时间的流逝当中，才能慢慢的发挥价值）

    # 思考一：人力资本是耐用品还是易耗品？

    回答：人力资源是一种耐用品

    我们每个人的人力资本也是耐用品，他必须在时间的维度上才能发挥他的功能，散发他的服务。一种耐用品，只有在时间的维度上才能发挥它的价值，才能散发它所蕴含的服务。

    任何一种能够带来收入的资源、本领或者禀赋，都是一种耐用品。


    # 思考二：儿童是否也是一种可消费的耐用品？

    回答：加里·贝克在一次演讲中说到：“儿童是一种可消费的耐用品”

    • 每个家庭生儿育女其实是理性的决定，生下这个小孩实际上就是一个消耗品，实际上就是一个长期投资。

    • 小孩儿很可爱，你只能在时间的流逝当中，去爱他、去欣赏他，去从中得到快乐。



    # 课后思考


    问：如果儿童是耐用品的话，那么为什么比较贫穷的社会，人们生小孩生得比较多？比较富裕的社会，人们生小孩反而生得少呢？

    # 我的思考：

    贫穷的社会里，人们认为生小孩会带来新的劳动力，可以创造更多的财富，所以生的比较多；

    富裕的社会里，由于接受教育，人们的思想发生了转变，生小孩会带来经济上的负担，会带来财富的损耗，所以生的少；


    # 薛兆丰老师的回答：

    其中的原因有五点：

    • 第一：存活率。贫穷的社会，婴儿夭折率很高，所以多生才能保证存活的数量；

    • 第二：贫穷社会中，孩子通常是体力劳动的补给。多生孩子，特别是男孩子，家庭就能够多增加劳动力的补给。而富于的社会力，孩子更大的机会时从事脑力劳动，所需的培养时间就比体力劳动长的多；

    • 第三：父母的时间成本。富于的社会里，父母的时间成本高，我们说生小孩是一种享受，付出的代价越高，需求就越低；

    • 第四：贫穷社会里，孩子通常是家长的储蓄。那样的社会里，市场通常不健全，人们得养儿防老，多几个小孩是增加储蓄的可靠性；

    • 第五：随着富裕程度的增加，孩子质量比数量更重要。质量上的提升带来的满足感，远远比数量上的提升所带来的满足感要大；
